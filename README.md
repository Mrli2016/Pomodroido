# vue-extension-template
`vue-cli3` + `webpack4` + `element-ui`  实现编译打包Chrome浏览器插件

## 环境安装
```
npm install
```

### 开发环境编译并热更新
```
npm run serve
or
npm run build-watch
```

### 生产环境打包
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

#### 实现功能
-   [x] 工作时间结束进行浏览页面背景色变化提示
-   [x] 设置工作时间跟休息时间
